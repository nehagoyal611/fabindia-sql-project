# Fabindia SQL Database Project

This repository contains the complete database design for Fabindia, including:

## Contents
- fabindia_table.sql – Full database schema (tables + relationships)
- fabindia_data.sql – Sample dataset with 10 rows per table
- eerdiagram_fabindia.mwb – ER Diagram (MySQL Workbench)

## Key Features
- Realistic entities based on Fabindia’s operations  
- 10 fully normalised tables (1NF–3NF)
- Clean Primary–Foreign key mapping
- Memory-optimized data types
- Online + Offline order support
- Store-product inventory mapping
- Supplier + Category linkage

## How to Use
1. Run `fabindia_table.sql` in MySQL Workbench  
2. Then run `fabindia_data.sql`  
3. Open the `.mwb` file for the ER diagram
